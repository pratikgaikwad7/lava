Managing Users

This interface allows you to manage all system users. You can:

  • View all user accounts and their passwords
  • Edit user roles and factory assignments
  • Change usernames and reset passwords
  • Delete user accounts
Password Visibility

Click the eye icon next to any password field to reveal the actual password.
